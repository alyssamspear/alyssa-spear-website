# Alyssa Spear - Personal Website

A beautiful, high-end, responsive minimalist single-page website. Crafted with ultra-premium typography, smooth animations, interactive elements, and custom dark mode features.

Designed to be ultra-fast, lightweight, and deployable to Vercel or GitHub Pages instantly with zero configuration.

## ✨ Features

- **Premium Typography**: Custom type curation with Google Fonts.
- **Interactive Spotlight**: Fluid cursor-tracking gradient illumination using standard performance requestAnimationFrame loops.
- **Micro-Animations**: Smooth slide-in animations on page load, a pulsing online status badge, and hover expansions.
- **Sparkle Interactions**: Click on the name to trigger a gorgeous physics-based sparkle explosion.
- **Ultra-Fast & SEO Optimized**: Zero external framework bloat, semantic HTML5 structure, and full responsiveness across all viewports.
- **Zero Configuration**: Ready to be uploaded to GitHub and linked directly to Vercel for instant live hosting.

## 🚀 Instant Deployment with Vercel

This website is structured to deploy out-of-the-box on Vercel:

1. Create a repository on **GitHub**, **GitLab**, or **Bitbucket** and push these files to it.
2. Go to [Vercel](https://vercel.com/) and log in with your Git provider.
3. Click **Add New** > **Project**.
4. Import your newly created repository.
5. Vercel will automatically detect the static project. Click **Deploy**.
6. That's it! Your site will be live on a production-ready global CDN in seconds.

## 🛠️ Local Development

To view the site locally:
- Simply open the `index.html` file in any modern web browser.
- Alternatively, if you use VS Code, you can click "Go Live" using the **Live Server** extension.
